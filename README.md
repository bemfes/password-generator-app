# 🔐 Генератор паролей

Веб-приложение для генерации надежных паролей с настраиваемыми параметрами. Пользователь может выбрать длину пароля, использовать строчные и заглавные буквы, цифры и специальные символы, а также увидеть надежность сгенерированного пароля и скопировать его одним нажатием.

🌐 [Демо проекта](https://bemfes.github.io/password-generator-app/)


## 🛠️ Стек

* React 
* TypeScript
* Zustand
* Tailwind CSS
* FSD
* Vite 
* Vitest 
* React Testing Library
* Сommitlint
* Husky
* lint-staged
* GitHub Actions

## 🎯 Функционал

* Генерация случайных паролей
* Настройка длины пароля
* Использование строчных и заглавных букв для пароля
* Включение цифр в пароль
* Добавление специальных символов в пароль
* Копирование пароля в буфер обмена
* Отображение надежности пароля

## 🚀 Установка и запуск

1. Клонируйте репозиторий

```
git clone https://github.com/bemfes/password-generator-app.git
```

2. Установите зависимости
```
npm install
```

3. Запустите проект в режиме разработки
```
npm run dev
```
